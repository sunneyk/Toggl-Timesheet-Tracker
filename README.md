# How to Run track.py

## Downloading or Cloning the Repository

### Option 1: Download ZIP

1. Click on the green "Code" button located above the file list.
2. Select "Download ZIP" from the dropdown menu.
3. Save the ZIP file to your preferred location on your local machine.
4. Extract the contents of the ZIP file to a directory of your choice.

### Option 2: Clone with Git

1. Open your terminal or command prompt.
2. Navigate to the directory where you want to store the repository.
3. Run the following command to clone the repository:

```bash
git clone https://github.com/sunneyk/Toggl-Timesheet-Tracker.git
```

4. The repository will be cloned to your local machine.

## Running track.py

1. Open a terminal or command prompt.
2. Navigate to the directory where you downloaded or cloned the repository (i.e., the location of `track.py`).
3. Make sure you have Python installed. If you don't have it, download and install it from the official website: [Python Download Page](https://www.python.org/downloads/).
4. Run the following command to execute the `track.py` script:

```bash
python track.py
```

## Obtaining Your Toggl API Key

1. Log in to your Toggl account at [https://track.toggl.com](https://track.toggl.com).

2. Click on your profile icon at the bottom left corner of the page.

3. From the dropdown menu, select "Profile settings".

4. Scroll down to the "API Token" section at the bottom of the page.

5. Your API token will be displayed here. You can click on the "Reset" button if you want to generate a new API token.

6. Copy this API token and paste it into the `track.py` script where required, or keep it handy to enter into the program when prompted.

Remember to keep your API token secure, as it provides access to your Toggl data. Do not share it with others or expose it in publicly accessible locations.

Now you should be able to run the `track.py` script with your Toggl API key. If you encounter any issues, please refer to the repository's README or open an issue on the repository's page. Happy tracking!
